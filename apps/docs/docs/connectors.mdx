# Connectors

Below is a manually curated list of connectors that are built for Connery and publicly available.

Read [core concepts](/docs/introduction/core-concepts#connector) to understand what a connector is.
In case you want to create your own connector, check out our [create connector](/docs/guides/create-connector) guide.
If you want to add your connector to this list, please open a pull request.

| -------------------------------- | ------------------------------------------------------------------------------------------------------------- |

| OpenAI                           | [connery-io/openai](https://github.com/connery-io/openai)                                                     |
