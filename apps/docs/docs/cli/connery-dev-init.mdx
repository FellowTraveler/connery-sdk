# `connery dev init`

Create all the necessary files and folders to start developing a new plugin.

Run this command at the root of your project.
You will be asked a few questions to configure the plugin, and then the CLI will create the necessary files and folders.

<Tabs>
  <TabItem value="npx" label="npx" default>

```
npx connery@latest dev init
```

  </TabItem>
  <TabItem value="npm" label="npm">

```
connery dev init
```

  </TabItem>
